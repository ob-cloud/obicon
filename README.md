# obicon

iconfont of on-bright

通用字体图标，整合于iconfont

## 结构

```text
|++ docs
|++ src
|++ source
|- build.js
```

| 文件   | 描述                   |
| ------ | ---------------------- |
| docs   | 在线使用说明文档       |
| src    | 源文件                 |
| source | 从iconfont导下来的文件 |

## 构建

```shell
npm run build
```

## 部署

```shell
npm run deploy
```
